2. Daten in der App

Die App Kids Bucket List speichert alle eingegebenen Daten (Aktivitäten, Kinderprofile, Kategorien, Bewertungen) ausschließlich lokal auf dem Gerät des Nutzers mittels SwiftData.

  • Es werden keine Daten an externe Server übermittelt.
  • Es wird kein Benutzerkonto benötigt.
  • Es findet keine Cloud-Synchronisierung statt.
  • Die App nutzt keine Analyse- oder Tracking-Dienste.
  • Es werden keine Werbedaten erhoben.

3. Externe APIs

Für den optionalen Import von Aktivitäten können folgende externe Dienste abgefragt werden:

  • Google Books API – für die Buchsuche
  • OMDb API – für die Film- und Seriensuche
  • IGDB API – für die Spielesuche

Diese Abfragen erfolgen nur auf ausdrücklichen Wunsch des Nutzers (aktive Suche). Dabei werden ausschließlich die eingegebenen Suchbegriffe übermittelt. Es werden keine personenbezogenen Daten weitergegeben.
