2. Data in the App

Kids Bucket List stores all entered data (activities, child profiles, categories, ratings) exclusively on the user's device using SwiftData.

  • No data is transmitted to external servers.
  • No user account is required.
  • There is no cloud synchronisation.
  • The app does not use any analytics or tracking services.
  • No advertising data is collected.

3. External APIs

For the optional import of activities, the following external services may be queried:

  • Google Books API – for book search
  • OMDb API – for movie and series search
  • IGDB API – for game search

These requests are made only at the explicit request of the user (active search). Only the entered search terms are transmitted. No personal data is shared.
