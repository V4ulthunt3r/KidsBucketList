2. Data in the App

Kids Bucket List stores all entered data exclusively on the user's device using SwiftData.

  • No data is transmitted to external servers.
  • No user account is required.
  • There is no cloud synchronisation.
  • The app does not use any analytics or tracking services.
  • No advertising data is collected.

3. External APIs

For the optional import of activities, the following external services may be queried:

  • Open Library data catalog – for book search through a static catalog file hosted on GitHub Pages
  • OMDb API – for movie and series search
  • IGDB API – for game search

These requests are made only at the explicit request of the user. Only the entered search terms are transmitted. No personal data is shared.

4. This Website (GitHub Pages)

This landing page is hosted via GitHub Pages. When you visit this page, GitHub automatically collects server log files which may include your IP address. For more information, please see GitHub's Privacy Statement.

We do not use any cookies, tracking or analytics tools on this website.

5. Your Rights

Since we do not collect or process any personal data, there are no specific data subject rights to assert against us. If you have any questions, please contact peter@hildel.com.
